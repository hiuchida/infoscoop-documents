# メッセージコンソールを表示する

コマンドパネル上のアイコン(![情報アイコン][Information icon]、![警告アイコン][Warn icon]または![エラーアイコン][Error icon])をクリックすると、メッセージコンソールを表示します。異常が発生した場合はメッセージコンソールを開き、発生しているエラーを確認し、システム管理者に連絡してください。

![メッセージコンソールの画面][Message Console]
